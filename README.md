# Portus Cart for WooCommerce

Portus Cart for WooCommerce is a responsive AJAX side cart for WooCommerce stores.

It helps customers review cart items, update quantities, apply coupons, remove products, and continue to checkout without leaving the current page.

[WordPress.org Plugin](https://wordpress.org/plugins/portus-cart-for-woocommerce/) | [Product Page](https://useportus.com/cart) | [Documentation](https://useportus.com/docs/portus-cart) | [Support](mailto:plugins@useportus.com)

## Features

- Responsive side cart panel for WooCommerce.
- Optional floating cart button with item counter.
- Automatic opening after WooCommerce AJAX add to cart.
- Quantity updates without page reloads.
- Coupon application and removal using WooCommerce settings.
- AJAX item removal.
- Checkout and cart page links.
- Low stock alerts.
- Admin settings for labels, colors, positioning, and compatibility.
- Scoped frontend styles to reduce theme conflicts.

## Requirements

- WordPress 6.0 or newer.
- PHP 7.4 or newer.
- WooCommerce active.
- A theme that prints the standard WordPress footer hook.

## Installation

The recommended installation method is through WordPress.org:

1. Open **Plugins > Add New** in WordPress.
2. Search for **Portus Cart for WooCommerce**.
3. Install and activate the plugin.
4. Open the Portus Cart settings page.
5. Configure the cart title, floating button, colors, and compatibility options.
