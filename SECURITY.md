# Security Policy

If you believe you have found a security issue in Portus Cart for WooCommerce, please do not open a public issue.

Email the maintainer at plugins@useportus.com with:

- a short description of the issue;
- steps to reproduce;
- affected version;
- WordPress, WooCommerce, and PHP versions;
- any relevant screenshots or logs.

We will review the report and respond as soon as possible.
